# odin-recipes
Basic html project on recipes made for The Odin Project.
## This project is a simple multi-page recipe website built using HTML. It includes a main page that links to three individual recipe pages. Each recipe page contains an image of the dish along with both unordered and ordered lists to present ingredients and step-by-step instructions clearly.

## The purpose of this project is to practice structuring content using HTML and to build a basic multi-page website with navigation between pages.

# Project Description

The website includes:

- A main homepage listing 3 recipes
- 3 separate HTML pages for each recipe
- Images representing each dish
- Unordered lists for ingredients
- Ordered lists for cooking steps
- Basic navigation between pages using links

# Skills Demonstrated


- Creating and structuring HTML documents
- Using headings, paragraphs, and HTML elements
- Building ordered and unordered lists
- Adding images to web pages
- Linking multiple HTML pages together
- Organizing a multi-page website structure
- Basic web design and content layout
